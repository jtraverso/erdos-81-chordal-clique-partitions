# Build environment

The source is designed for XeLaTeX and uses standard packages including:

- `amsmath`, `amssymb`, `amsthm`, `mathtools`;
- `geometry`, `microtype`;
- `enumitem`, `booktabs`, `tabularx`;
- `xurl`, `hyperref`, `bookmark`;
- `unicode-math`, `fontspec` through XeLaTeX.

A current TeX Live or MiKTeX installation with these packages is sufficient.
The official release PDF contains 13 pages.
