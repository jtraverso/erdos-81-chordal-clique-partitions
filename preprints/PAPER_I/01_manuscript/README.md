# Manuscript

This directory contains the three official representations of the same release:

- `PAPER_I_preprint_v1.0.pdf` — publication-facing document;
- `PAPER_I_preprint_v1.0.tex` — typesetting source;
- `PAPER_I_preprint_v1.0.md` — accessible Markdown source.

These files form one release set. Changes to any one of them require a new version
and a renewed cross-format consistency check.
