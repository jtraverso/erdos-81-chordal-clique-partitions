# Manuscript

Official representations of the same release, English and Spanish:

- `PAPER_II_preprint_v1.0.pdf` / `_es.pdf` -- publication-facing documents;
- `PAPER_II_preprint_v1.0.tex` / `_es.tex` -- typesetting sources;
- `PAPER_II_preprint_v1.0.md`  / `_es.md`  -- accessible Markdown sources;
- `figures/` -- figures referenced by the `.tex`/`.md` sources.

The English and Spanish versions share identical mathematics: every displayed
equation, the theorem statement, and the formal-verification claims are the same;
only the prose is translated.
