# Erdos Problem #81 - Chordal Clique Partitions

This repository contains the public manuscript sequence and proof roadmaps for an announced resolution of Erdos Problem #81, originally posed by Erdos, Ordman, and Zalcstein, on the edge clique partition number of chordal graphs.

## Announced Result

The announced asymptotic result is

```text
max{cp(G): G chordal, |V(G)| = n} = (1/6 + o(1)) n^2.
```

Equivalently, every chordal graph G on n vertices satisfies

```text
cp(G) <= (1/6 + o(1)) n^2.
```

The constant 1/6 is sharp, already for the complete split graph

```text
K_p join overline{K}_{2p}, where n = 3p.
```

## Proof Strategy

The announced proof proceeds through a fractional triangle-packing theorem for chordal graphs:

```text
|E(G)| - 2 nu_3^*(G) <= (1/6 + o(1)) n^2.
```

Together with standard fractional-to-integral triangle-packing rounding for the fixed family {K_3}, this implies the announced chordal clique partition bound.

The full technical proof is organized as a four-part manuscript sequence.

## Proof Roadmaps

The folder

```text
docs/proof-roadmaps/
```

contains the global proof map, dependency discipline, calculation ledgers, certificate usage notes, reviewer-sensitive points, and the v0.2-to-v0.3 correction record.

These files are intended to make the proof chain auditable without requiring readers to inspect the historical gate notes.

## Historical Sealed Archive

Earlier technical drafts for Papers B, C, and D were archived in encrypted form in:

```text
sealed/papers_B_C_D_technical_drafts_v02_encrypted.zip.enc
```

This file was generated by encrypting a local plaintext ZIP bundle with OpenSSL AES-256-CBC using PBKDF2. It is retained as a timestamped historical milestone.

The SHA-256 hashes are recorded in:

```text
manifest_sha256.txt
```

The corresponding manuscripts are now posted openly in the paper-specific folders.
