# Final v0.3 Public Status

This note records the public status of the v0.3 repository organization.

The A-D manuscript sequence is posted openly in paper-specific folders:

```text

paper_B/
paper_C/
paper_D/
```

The v0.3 versions incorporate the reviewer-sensitive corrections recorded in `v02_to_v03_reviewer_corrections.md` and the proof-roadmap documentation in this folder.

The proof-roadmap folder is intended to replace the need to inspect historical gate notes for the formal proof chain.

The public proof chain is:

```text
Paper A -> Paper B -> Paper C -> Paper D
```

with each paper citing only earlier papers or standard external results.

## Historical notes
