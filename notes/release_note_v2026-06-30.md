# Release Note - v2026-06-30-announcement

This release records a timestamped milestone for the announced asymptotic resolution of Erdos Problem #81 on clique partitions of chordal graphs.

The announced result is:

```text
max{cp(G): G chordal, |V(G)| = n} = (1/6 + o(1)) n^2.
```

Included public artifacts:

- Research announcement draft v0.1.
- Paper A preprint draft v0.2 on split graphs.
- SHA-256 manifest.
- Encrypted archive of Papers B, C, and D technical drafts.

The encrypted archive is included for timestamping and priority purposes. The decryption key will be released after the corresponding manuscripts are publicly posted.

This release is not a peer-reviewed publication.
